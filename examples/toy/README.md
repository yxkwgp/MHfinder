# Toy example files

This directory contains synthetic files for format demonstration and lightweight parser tests. They are intentionally small and do not require PLINK.

Files:

- `sample_info.tsv`: required sample metadata format.
- `candidate_MHs.txt`: candidate-MH file format produced by `prep-MHs`.
- `selected_MHs_example.txt`: selected-MH file format produced by `screen-MHs-by-SCGWAS`.

These files are not a biological benchmark and should not be used to evaluate ancestry-inference accuracy.
